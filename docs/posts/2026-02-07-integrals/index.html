Sigmoid Function

The sigmoid function \(\sigma(t) = \frac{1}{1+e^{-t}}\) is widely used in machine learning. Its integral is \(\int_a^b \sigma(\tau) \, d\tau = \ln(1+e^b) - \ln(1+e^a)\).
